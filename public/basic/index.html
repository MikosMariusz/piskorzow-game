O Grze

To jest przykład prostej strony internetowej utworzonej tylko za pomocą HTML, CSS i JavaScript.

Nasza gra terenowa to interaktywna przygoda, która łączy prawdziwe miejsca z wirtualnymi wyzwaniami!

Jak to działa?

  • HTML - struktura strony (nagłówki, teksty, obrazki)
  • CSS - wygląd strony (kolory, rozmiary, efekty)
  • JavaScript - interakcje (kliknięcia, animacje)
